Deployment
1. Keep index.html in the project root.
2. Keep the calculator page inside /calculator/index.html.
3. Zip the full folder contents so index.html is at the top level of the ZIP.
4. Upload the folder contents to Netlify, or drag and drop the ZIP after confirming the root contains index.html.

Integration note
Replace the styled placeholder area inside /calculator/index.html with the existing calculator markup and logic.
Keep the page shell, header, footer, shared style.css, and shared script.js.
